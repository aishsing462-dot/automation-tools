# Personal Automation Tools

This repository contains scripts to automate various tasks on your Windows PC.

## Structure

- **file_management/**: Scripts to organize files (e.g., sort Downloads).
- **web_scraping/**: Browser automation and data extraction.
- **system_tasks/**: Maintenance and scheduling scripts.
- **coding_workflows/**: Git hooks and development helpers.

## Setup

1. Install Python 3.12+.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
